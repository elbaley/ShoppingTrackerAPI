# TechCareer .NET Core Foundations Bootcamp Final Project
## Shopping Tracker (ASP.NET Backend with React.js Frontend)


### Backend 
- Postgresql Database
- JWT authentication
- Swagger

### Frontend
- React Context
- React Router DOM
- Tailwind

### How To Build
- Backend:
```shell
cd ShoppingTrackerAPI
dotnet run
```

- Frontend:

```shell
cd frontend
yarn
yarn dev
```
